# TaskFlow - Quản Lý Công Việc

Ứng dụng theo dõi công việc hàng ngày, tuần, tháng, năm. Chạy hoàn toàn trên trình duyệt, dữ liệu lưu cục bộ (localStorage).

## Tính năng
- Xem theo Ngày / Tuần / Tháng / Năm
- Thêm, sửa, xoá, đổi mức ưu tiên công việc
- Tìm kiếm công việc
- Chế độ tối / sáng
- Sao lưu & khôi phục dữ liệu (.json)

## Deploy bằng GitHub Pages
1. Vào **Settings** > **Pages**
2. Source: chọn nhánh `main`, thư mục `/ (root)`
3. Save, đợi 1-2 phút, truy cập link được cấp
